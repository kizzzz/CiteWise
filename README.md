# CiteWise V3 — 智能研究助手

基于 LLM + RAG 的学术论文研究助手，支持：

- **PDF 上传与解析** — 自动提取文本、表格、图表元数据
- **混合检索** — 向量检索 + BM25 + RRF 融合 + 重排序
- **多 Agent 协作** — Router/Researcher/Writer/Analyst/Coordinator
- **三色来源标注** — 📖文献(蓝) / 🌐联网(绿) / 🧠推理(紫)
- **结构化提取** — 自定义字段提取 + Excel导出 + 可视化图表
- **联网搜索** — DuckDuckGo API 实时信息检索
- **章节管理** — 主对话 + 子对话协同写作
